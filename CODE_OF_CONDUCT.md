# Code of Conduct

## Our Standards

We are committed to a respectful, welcoming, and harassment-free community.

Examples of positive behavior:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the community and project

Examples of unacceptable behavior:

- Harassment, insults, or discriminatory language
- Trolling, personal attacks, or inflammatory comments
- Publishing private information without consent

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, and discussions.

## Reporting

Report unacceptable behavior privately to project maintainers via GitHub contact channels.
